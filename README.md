# Frontend

This is a JavaScript frontend application that connects to an educational backend API. The backend is provided separately for learning purposes.

## Development server

Run `npm run` dev for a frontend development server. The application will typically run on `http://localhost:5173` (or as configured). The app will automatically reload if you change any source files. change any of the source files.

## Backend Server

To run the educational backend server (provided separately), navigate to the backend directory and run `npm start`. The backend runs on `http://localhost:3000` or a similar port.
## Project Structure

/src - Frontend source code (JavaScript/TypeScript, components, styles)

/public - Static assets

Configuration files (package.json, etc.)

## Important Notes

This repository contains only the frontend code

The backend is an external educational API provided for learning purposes

Ensure both frontend and backend servers are running for full functionality

The frontend is configured to communicate with the backend API endpoint
